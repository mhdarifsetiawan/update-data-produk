## Prerequisites

### Requirements

1. NodeJS v18.20.3 or later

### Install

```
$ npm install
```
### Running App

```
npm run gas
```
OR
```
npm run build
npm run start
```

## Configuration

### Env File

| Code                  | Description                           | Default Value                      |

| DB_PORT                  | -                  | 5432                               |

| DB_USER               | -                       |                                    |
| DB_PASSWORD               | -                   |                                    |
